本仓库是主包大二上的一次离散实验作业，仅作为一次记录。
仓库名13579来源于我对“离散”这个术语的主观的、狭义的理解：在笛卡尔坐标系上几个纵坐标未知但横坐标不同且并不无限接近的点。
主包大一的实验作业没啥意思，全让ai代理了，本来这次大二的第一次实验作业我也是准备ai代理的，但是离散老师（李进凯，凯哥）的一句话把我点醒了，以后面试找工作是一定会让我当场写代码的。
虽说现在有ai这个工具来辅助写代码，但长远看来，ai并不会提升我的代码能力，这个仓库，于我而言就算是一次尝试、一次挑战吧。

此README文件为一次代码复盘，分析自己在编写代码时遇到的问题。

代码的要求是用自己熟悉的程序设计语言实现对一个命题公式的：

（1）真值表的打印
（2）公式类型的判断
（3）两种主范式的求解

抛开代码运行环境这个繁琐、暧昧的问题，我在编程过程中遇到的最困难的问题就是把问题拆解成几个部分来实现，其次是分析要使用的数据结构。



提及本次作业的代码运行环境，我使用的是自以为最熟悉（仅仅是安装了许多插件，为IDE配置了一个图片背景，亲手过写代码的次数屈指可数）的VsCode IDE。
